# Installing and using the components in your project

## Using the components

1. Run `npm install @ukic/react` at the root level of your project.

2. Add `import { IcComponent } from "@ukic/react"` to use the relevant component.

### Importing CSS styles

To use these components with the correct css styles, import the core CSS file into your project:

1. Add `@import "@ukic/web-components/dist/core/core.css"` at the top level CSS file for your project.

### Testing

The `@ukic/react` package will need to be transformed before you can use these components in Jest tests:

1. Add a `transformIgnorePatterns` field with the value `["/node_modules/(?!@ukic/react)"]` to your Jest config.
